# Resolución de Backtracking Micromouse

## Descripción

Proyecto universitario enfocado en la implementación del algoritmo de búsqueda Backtracking para resolver un laberinto tipo Micromouse.

El objetivo es encontrar una ruta válida desde el punto inicial hasta la meta mediante exploración de caminos y retroceso cuando se encuentran rutas bloqueadas.

## Algoritmo utilizado

- Backtracking recursivo
- Exploración de celdas
- Registro de posiciones visitadas
- Retroceso ante caminos inválidos

## Funcionamiento

El algoritmo analiza las posibles direcciones disponibles dentro del laberinto:

1. Selecciona un camino posible.
2. Avanza hacia una nueva celda.
3. Marca la celda como visitada.
4. Si encuentra un bloqueo, regresa a la posición anterior.
5. Continúa buscando hasta encontrar la solución.
